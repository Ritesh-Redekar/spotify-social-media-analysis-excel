# Spotify Social Media Data Analysis Project

## Overview
This project analyzes Spotify's social media performance using Microsoft Excel.  
The objective was to clean the dataset, analyze engagement metrics, evaluate campaign performance, and understand follower behavior.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Charts & Data Visualization
- Data Analysis

## Project Tasks

### Task 1 – Data Cleaning
Cleaned and organized the raw dataset by removing duplicates, correcting formatting issues, and preparing the data for analysis.

### Task 2 – Engagement Analysis
Analyzed likes, comments, and shares to understand which content types generate the highest engagement.

### Task 3 – Platform Performance
Compared engagement across different social media platforms to determine which platform performs best.

### Task 4 – Hashtag & Content Strategy
Evaluated hashtag usage and content types to identify strategies that increase visibility and engagement.

### Task 5 – Campaign Effectiveness
Analyzed marketing campaign performance based on engagement metrics and audience interaction.

### Task 6 – Follower Retention & Loyalty
Examined follower behavior and engagement trends to understand retention and loyalty patterns.

## Key Insights
- Certain content types generate significantly higher engagement than others.
- Specific hashtags help improve visibility and reach on social media platforms.
- Campaign performance varies across platforms depending on audience interaction.
- Consistent posting and engaging content improve follower retention and loyalty.
